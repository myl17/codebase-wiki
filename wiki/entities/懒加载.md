# 懒加载

模块或资源在真正被使用时才加载，而非启动时全量加载。典型实现：Promise 缓存包装动态 import、按需 require、条件初始化。

## 在各 repo 中的体现

- [[openclaw/dimensions/openclaw-performance-tradeoffs]] — `createLazyRuntimeModule` 用 `cached ??= importer().then(select)` 包装动态 import，TTS / media-understanding / model-auth 等重型模块全部延迟
- [[openclaw/dimensions/openclaw-extension-points]] — `defineBundledChannelEntry` 让 channel plugin 代码只在该 channel 被使用时加载
