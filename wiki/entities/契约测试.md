# 契约测试

验证接口实现方是否符合约定协议的测试。通过共享 test suite 定义（而非重复手写），新实现注册后自动被覆盖，避免接口漂移。

## 在各 repo 中的体现

- [[openclaw/dimensions/openclaw-testing-philosophy]] — `installChannelActionsContractSuite(...)` 定义共享 contract test suite，新 channel 注册到 registry 后自动覆盖；`forks` pool + `isolate: false` 共享注册表状态
